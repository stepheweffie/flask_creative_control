LIGHTSTREAMER - ADAPTER REMOTING INFRASTRUCTURE RESOURCES
=========================================================

This directory includes the resources needed to deploy the proxy adapters
to enable the server to connect with remote Data Adapters and Metadata Adapters.

- Version.txt
  Reports the current version of the Adapter Remoting Infrastructure.

- doc/Adapter Remoting Infrastructure.pdf
  Contains an overview of the Adapter Remoting Infrastructure and deploy instructions
  for the Proxy Adapters.
  
- doc/remote_adapter_conf_template/adapters.xml
  A sample configuration file for the Proxy Adapters.

- doc/remote_adapter_robust_conf_template/adapters.xml
  A sample configuration file for the "Robust" version of the
  Proxy Adapters.
